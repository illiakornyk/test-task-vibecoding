# DB2 Projects Collection

This repository contains a collection of full-stack and mobile AI-powered applications.

## 🚀 Projects

### 🎙️ [Voice-to-Text SaaS](./full_stack_app/README.md)
A powerful audio processing and transcription platform built with **Next.js 15**, **Clerk**, **Stripe**, and **Prisma**.
- **Key Features**: Transcription, secure authentication, recurring subscriptions, and AI-powered text processing.
- [Read the full documentation here](./full_stack_app/README.md)

### 💬 [Qwen Mobile Chat](./qwen-mobile-chat/README.md)
A high-performance, dual-mode React Native chat application with support for both cloud and local on-device inference.
- **Key Features**: Local LLM (Qwen 0.5B), zero-dependency UI, and native markdown rendering.
- [Read the full documentation here](./qwen-mobile-chat/README.md)
